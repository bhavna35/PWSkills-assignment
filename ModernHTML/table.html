HTML vs HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 stands for HyperText Markup Language version 5.
HTML 4.01 was the last major version before HTML5. HTML5 is the latest major version of HTML.
HTML 4.01 had limited support for multimedia and interactive elements. HTML5 provides native support for audio, video, and canvas, allowing more multimedia and interactivity.
HTML 4.01 did not have the semantic elements introduced in HTML5. HTML5 introduced semantic elements like <header>, <nav>, <section>, and <footer> to improve document structure and accessibility.
