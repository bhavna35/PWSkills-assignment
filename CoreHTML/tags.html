List of HTML Tags

  1. <html>: The root element that wraps all content on the web page.
  2. <head>: Contains meta-information about the document, such as the title, character set, and linked stylesheets.
  3. <title>: Sets the title of the web page, which appears in the browser's title bar or tab.
  4. <meta>: Provides metadata about the document, including character encoding and viewport settings.
  5. <link>: Specifies external resources like stylesheets, typically used to link to CSS files.
  6. <style>: Defines inline CSS styles or includes CSS code within the HTML document.
  7. <script>: Embeds JavaScript code within the HTML document or references external script files.
  8. <body>: Contains the visible content of the web page, such as text, images, and other media.
  9. <header>: Represents a container for introductory content or a set of navigational links at the top of a page.
  10. <nav>: Defines a section of navigation links, often found in menus or navigation bars.
  11. <main>: Indicates the main content of the web page, and there should be only one <main> per page.
  12. <article>: Represents a self-contained piece of content, such as a blog post or news article.
  13. <section>: Divides content into thematic sections, helping with document structure.
  14. <aside>: Contains content that is tangentially related to the surrounding content, like sidebars.
  15. <footer>: Represents the footer of a document or a section, typically containing copyright information and links.
  16. <div>: A generic container used for grouping and styling elements or creating layout structures.
  17. <p>: Defines a paragraph of text.
  18. <a>: Creates hyperlinks to other web pages or resources.
  19. <img>: Embeds images on the page.
  20. <ul>: Defines an unordered list, typically used with <li> elements to create lists.
  21. <ol>: Defines an ordered (numbered) list, also used with <li> elements.
  22. <li>: Represents a list item within <ul> or <ol> lists.
  23. <table>: Defines a table, commonly used for displaying data in rows and columns.
  24. <tr>: Represents a table row within a <table>.
  25. <td>: Defines a table cell within a <tr>.
  26. <form>: Creates a form for user input, used to gather data from users.
  27. <input>: Specifies input fields within a form, like text boxes or buttons.
  28. <textarea>: Defines a multi-line text input field within a form.
  29. <select>: Creates a dropdown list within a form.
  30. <button>: Defines a clickable button within a form or web page.
